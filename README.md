# 英语听说训练 · English Listening & Speaking Trainer

一个帮助快速提升英语**听力**和**口语**的手机网页 App（PWA）。纯前端、无需服务器、无需安装构建工具，iPhone / 安卓用浏览器打开即可"添加到主屏幕"当 App 用。

## 功能

| 模块 | 说明 | 是否需要联网/付费 |
|------|------|------------------|
| 🎧 **听力精听** | 逐句播放标准发音，可调语速，英文 / 音标 / 中文字幕随意开关，点单词可单独发音 | 免费 · 可离线 |
| 🗣️ **影子跟读** | 听标准音 → 按住录音跟读 → 自动语音识别打分 + 回放对比 | 免费（打分用浏览器内置识别） |
| ✍️ **精听听写** | 听一句写下来 → 逐词对照高亮对错 | 免费 · 可离线 |
| 🤖 **AI 口语陪练** | 和 AI 外教用英语对话，自动纠正语法与表达，朗读回复练听力 | 需联网 + API Key |

> 学习建议：**影子跟读** 是提升口语最快的方法，**精听听写** 是提升听力最快的方法。每天各练 10 分钟，比刷大量视频更有效。先盲听，听不懂再开字幕。

## 如何使用

### 在电脑上预览
因为用到麦克风和 Service Worker，需要通过本地服务器打开（不能直接双击 html）：

```bash
# 任选一种
python -m http.server 8000
# 或
npx serve .
```

然后浏览器打开 http://localhost:8000

### 在 iPhone 上当 App 用（无需 Mac、无需上架）
1. 把这些文件部署到任意静态托管（GitHub Pages / Cloudflare Pages / Vercel 等，免费）
2. iPhone 用 **Safari** 打开网址
3. 点底部「分享」→「添加到主屏幕」
4. 桌面会出现图标，点开即全屏，跟原生 App 一样

> 这种方式**不会 7 天过期**、**不需要苹果开发者账号**，最适合自己使用。

## 配置 AI 外教

进入「设置」，选择服务商并填入 API Key（只保存在本机浏览器，不上传）：

- **Claude**（英语质量最佳）：到 console.anthropic.com 申请。默认模型 `claude-haiku-4-5`（快又便宜）。
  ⚠️ Claude **MAX 订阅不能用于 API**，需在控制台单独充值额度。
- **DeepSeek**（便宜、国内直连、支持国内支付）：到 platform.deepseek.com 申请。默认模型 `deepseek-chat`。

两者随时可在设置里切换。个人练口语每天通常仅几美分到几毛钱。

## 技术说明

- 朗读 = 浏览器 `SpeechSynthesis`，识别/打分 = `SpeechRecognition`（iOS Safari、Chrome 支持较好）
- AI 对话从浏览器直接调用厂商 API（个人使用无需后端）；若遇跨域限制，可后续加一个轻量代理
- 离线由 Service Worker 缓存应用外壳实现；AI 对话仍需联网

内置对话均为**原创**（日常、餐厅、出行、职场），无版权问题。后续可扩展为导入你**合法拥有**的文本，或贴 YouTube 链接拉取官方字幕学习——请勿抓取下载有版权的影视资源（如《老友记》等）。
